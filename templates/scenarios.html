Scenario A: Basic Deployment

Create a Protection Domain with 3 SDS nodes, a storage pool with 2-copy protection, 2 SDC clients, and a 500GB thin volume mapped to SDC1.

Scenario B: Failover Test

Start with Scenario A setup, then fail an SDS node to test pool degradation, failover, and rebuild.

Scenario C: Multi-Host Mapping

Map the same volume to multiple SDC clients to demonstrate read/write access modes and concurrent IO handling.

Scenario D: Capacity Exhaustion

Fill the pool close to capacity, then attempt to create additional volumes to demonstrate insufficient capacity handling.
